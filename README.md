# CardMap（カードマップ）

手書きメモアプリケーション - カード形式でメモを整理・管理できるWebアプリケーション

## 概要

CardMapは、カード形式でメモを自由に配置し、接続線で関連付けできる手書きメモアプリケーションです。思考整理やアイデア出しに最適です。

## 機能

- 📝 カード形式のメモ作成・編集
- 🔗 カード間の接続線による関連付け
- 🎨 カードの色やサイズのカスタマイズ
- 📊 様々な思考整理テンプレート（マインドマップ、KWLチャート、PMI、ベン図など）
- 💾 ローカルストレージへの自動保存
- 📥 画像としてエクスポート機能
- 🔍 ズーム・パン機能

## 使い方

1. `index.html`をブラウザで開く
2. カードを追加してメモを記入
3. カードをドラッグして配置
4. カード間を接続して関連付け
5. 必要に応じて画像としてエクスポート

## 技術スタック

- HTML5
- CSS3
- JavaScript（Vanilla）
- Tailwind CSS（CDN）
- html2canvas（画像エクスポート用）
